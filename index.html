JS and JQuery Flashcards

JavaScript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A symbol that is used to perform an operation.

Variables

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Variable naming conventions

A set of rules for choosing the character sequence to be used for identifiers which denote variables.

Functions

a block of code that performs an action and returns a result; optionally takes arguments.

Methods

Actions that can be performed on objects.

Arguments

A specific input in the function, also known as an independent variable.

Parameters

A variable that's assigned to the argument.

Return

Result of a function.

Chaining methods

Repeatedly calling one method after another on an object, in one continuous line of code.

Strings

Any finite sequence of characters

Booleans

A binary variable, having two possible values called “true” and “false.”

Undefined

An object (such as a variable) without a defined value.

NaN

Not a number.

Escape

Encodes a string.

Alert

Opens a dialog box and returns undefined.

Comments

Place two slashes "//" in front of the code or text you wish to have the JavaScript interpreter ignore.

jQuery

A concise and fast JavaScript library

Attributes

A specification that defines a property of an object, element, or file.
